
# Robotos Clone Landing Page

This project is a landing page built with HTML and CSS, inspired by the design and user experience of the Robotos website (https://www.robotos.art/).

🚀 About the Project

The purpose of this project is to recreate the structure, layout, and visual elements of the original website as a front-end development study, focusing on responsive design and best practices for building web interfaces.

🎯 Learning Objectives

This project was developed to practice:

Front-end development fundamentals;
Responsive web design;
UI recreation and layout structuring;
Clean and maintainable code organization.
